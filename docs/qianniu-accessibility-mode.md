# 千牛无障碍模式开启说明

## 先区分两个概念

这里说的“千牛无障碍模式”，一般指千牛 PC 客户端内部为了多账号/自动化/辅助工具暴露界面元素的模式，不是 Windows 系统设置里的“辅助功能”。

部分千牛自动化项目明确要求：

- 使用千牛多账号模式；
- 打开千牛无障碍模式；
- 千牛窗口保持前台可见，不能最小化。

## 常见开启路径

不同千牛版本菜单名称可能略有差异，按下面顺序找：

### 方案 A：从千牛主界面设置进入

1. 打开千牛 PC 客户端。
2. 使用“多账号模式”登录或切换到“多账号模式”。
3. 点击右上角或左下角的“设置 / 齿轮 / 三横线菜单”。
4. 进入“系统设置”。
5. 在左侧找下面类似栏目：
   - 接待设置
   - 聊天设置
   - 客服设置
   - 辅助功能
   - 无障碍设置
6. 找到并勾选：
   - 开启无障碍模式
   - 启用无障碍模式
   - 无障碍接待模式
7. 保存设置。
8. 完全退出千牛并重新打开。

### 方案 B：从多账号接待模式入口进入

有些版本只有进入多账号接待后才显示无障碍相关开关：

1. 打开千牛。
2. 切换到“多账号模式 / 多账号接待”。
3. 进入接待台。
4. 找接待台右上角设置按钮。
5. 在接待设置里搜索或查找“无障碍”。
6. 勾选后重启千牛。

## 验证是否开启成功

启动本项目后打开：

```text
http://127.0.0.1:8765
```

按顺序测试：

1. 点击“刷新窗口”。
2. 下拉框里应该能看到千牛窗口，并且分数较高。
3. 选择千牛窗口。
4. 点击“读取控件文本”。

如果日志里能看到类似：

- 买家昵称；
- 接待列表文字；
- 输入框；
- 发送按钮；
- 部分聊天区文本；

说明无障碍/UIA 暴露有效。

如果只能看到窗口壳子，看不到聊天内容，属于正常现象。千牛部分聊天区域是自绘控件，即使开启无障碍，也可能无法直接读到消息正文。这时需要继续使用：

- 剪贴板读取；
- 聊天区域截图；
- AI 视觉识别；
- OCR 兜底。

## 找不到无障碍模式怎么办

按下面顺序排查：

1. 确认是否是千牛 PC 客户端，不是网页版。
2. 确认是否使用多账号模式。
3. 在千牛设置搜索框里搜索“无障碍”。
4. 查看“接待设置 / 聊天设置 / 客服设置 / 辅助功能”。
5. 更新或切换千牛版本后再找。
6. 仍找不到时，可以临时打开 Windows 讲述人测试 UIA 暴露：

```text
Win + Ctrl + Enter
```

再次点击本项目里的“读取控件文本”。

注意：Windows 讲述人不是千牛无障碍模式，只是用于验证 UIAutomation 是否能读到更多控件。

## 对本项目的建议

稳定方案不是只依赖截图，也不是只依赖 UIA，而是三层混合：

```text
第一层：UIAutomation / 无障碍控件读取
第二层：Ctrl+A / Ctrl+C 剪贴板读取聊天记录
第三层：截图 + AI 视觉 / OCR 兜底
```

后续项目应把截图识别降级为兜底，把无障碍控件读取和剪贴板读取作为主链路。
